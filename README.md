# 🎬 Movie Watchlist App

A Flutter mobile app for browsing movies and managing a personal watchlist. Built with **Flutter/Dart** following a clean widget-based architecture.

---

## 📁 Project Structure

```
lib/
├── models/
│   ├── movie.dart              # Movie data model
│   └── watchlist_model.dart    # Watchlist state management
│
└── widgets/
    ├── detailsPage.dart        # Movie details screen
    ├── moviesdisplay.dart      # Movies listing/browse screen
    └── watchllist_page.dart    # Personal watchlist screen

main.dart                       # App entry point
pubspec.yaml                    # Dependencies & assets
```

---

## ✨ Features

- Browse a list of movies
- View detailed info for each movie
- Add/remove movies from a personal watchlist
- Dedicated watchlist page to track saved movies

---

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Architecture:** Widget-based with model separation

---

## ▶️ Getting Started

### Prerequisites

- Flutter SDK installed
- An emulator or physical device

### Run the app

```bash
flutter pub get
flutter run
```
